What is TGMacro?

TGMacro is one of the most popular and well-known player macro tools, which is also considered to be one of the safest. With TGMacro, you can set any keyboard or mouse keys to be triggered in the situations you want, and can free yourself from tedious tasks in games or daily computer use!









Okay but, How to use it?

One of the most beloved features of TGMacro is that it is very easy to learn. All you need to do is create a new macro and select which keys will be sent when you press a certain key. You don't even need to make these settings manually! Just select Add New Macro and let it record the keys you want to mimic. For more information, you can visit the How to use page.









Why I need TGMacro?

The primary target of TGMacro is gamers, which is why it is mostly used by gamers. However, it also has convenient features to facilitate daily use.

For example, some games like to make things difficult for users. Even for simple actions, you may need to press multiple keys in a row. So why not do these actions once and then have them repeated by pressing a single key? This is where TGMacro comes to your aid.

Or maybe you were given a tedious task at the office? All you have to do is press the same keys in the same order one by one? Why not let TGMacro do these things while you enjoy your coffee?









Why TGMacro?

Macros can be used for malicious purposes because they can read users' keystrokes. TGM is completely transparent and open source, so it is 100% reliable.

Moreover, it is completely free and yet has more features than many paid macro software on the market.

With TGM, you don't need to learn 'scripts' - you can create your macro just by using the interface, or you can just ask it to record.
